# study-webpack

Webpack은 모든 자산을 가져 와서 생산 준비가 완료된 번들로 출력합니다.

   - JavaScript 파일을 하나의 파일로 묶어서 HTTP 요청을 최소화합니다.
   - SASS / LESS 파일을 CSS로 처리하고 필요한 경우에만 사용하십시오.
   - JSX 또는 ES2015를 브라우저가 이해하는 바닐라 JS로 변환
